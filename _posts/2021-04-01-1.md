# Markdown Basic Syntax
------
## 1. 제목 (Headings)
제목을 만들기 위해서는 **원하는 문장 또는 글귀 앞에 # 을 넣어**주어야한다. 1개부터 6개까지의 # 개수로  크기 조절이 가능한데, # 의 개수가 적을 수록 더 큰 폰트로 제목 작성이 가능하다.

# 1. # 이 1개일 때
## 2. # 이 2개일 때
### 3. # 이 3개일 때
#### 4. # 이 4개일 때
##### 5. # 이 5개일 때
###### 6. # 이 6개일 때

## 2. 줄 바꿈 (Line Breaks)
줄을 바꾸기 위한 첫 번째 방법은 문장의 끝에 **space 를 두 번 사용**한 후 Enter 를 눌러 다음 문장을 작성하는 것이다. space 를 두 번 사용하지 않으면, Enter 를 눌러 작성하고 있는 Markdown 문서 내에서 줄 바꿈을 하더라도 실제로 보여지는 문서에서는 줄 바꿈이 되지 않은 채로 출력 된다.  

이외에도 HTML 태그인 **\<br> 을 사용**하여 줄 바꿈을 하는 방법이 있다.

## 3. 강조 (Emphasis)
강조를 하는 방법에는 **Bold 처리**를 하는 방법과 *Italic 처리*를 하는 방법이 있다.
### Bold
문장 또는 단어에 Bold 처리를 하기 위해서는 **원하는 문장 또는 단어 앞 뒤로 \* (asterisk) 를 두 개씩 삽입**하면 된다.
### Italic
문장 또는 단어에 Italic 처리를 하기 위해서는 **원하는 문장 또는 단어 앞 뒤로 \* (asterisk) 를 한 개씩 삽입**하면 된다.

Bold 와 Italic 처리를 **동시에** 하고 싶은 경우에는 **원하는 문장 또는 단어 앞 뒤로 \* (asterisk) 를 한 개씩 삽입**하면 된다.
